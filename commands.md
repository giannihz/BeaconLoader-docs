# Commands

## Main commands

### `/bloader give <player> <3x3|5x5|7x7>`
Gives a configured loader item.

### `/bloader reload`
Reloads the config and rebuilds runtime systems.

### `/bloader debug`
Displays current debug mode state.

### `/bloader tps`
Displays current server TPS.

### `/bloader tickets`
Shows loader ticket coverage information.

### `/bloader randomticks [world]`
Shows the random tick speed for a world.

### `/bloader loaderinfo`
Shows information about active loaders.

### `/bloader entityinfo`
Shows villagers, golems, and entity totals inside loader areas.

### `/bloader nms`
Shows basic server runtime environment information.
